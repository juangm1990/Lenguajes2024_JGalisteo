Resumen Tema 12: Conversión y adaptación de documentos XML.


XML es un formato de datos estructurado utilizado para almacenar y compartir información de manera jerárquica. Para modificar y transformar documentos XML, se emplean lenguajes como XSLT, que permite convertirlos en otros formatos, y XPath, que facilita la selección de elementos dentro del árbol XML. Además, existen otros formatos de intercambio de datos como JSON, CSV y YAML, cada uno con características específicas según su uso.

Lenguajes de Transformación XSL

XSL (eXtended Stylesheet Language) es un conjunto de tecnologías para transformar y dar formato a documentos XML. Su uso permite manipular la estructura del contenido de manera similar a cómo CSS estiliza documentos HTML.

  • XPath: Permite evaluar expresiones y localizar elementos dentro de un documento XML.
  • XSLT: Facilita la transformación de documentos XML en otros formatos, como HTML o XML con diferente estructura.
  • XSL-FO: Se emplea para generar documentos en otros formatos, como PDF, aplicando reglas de formato avanzadas.

¿Para qué sirve XPath?

XPath es un lenguaje que permite navegar y seleccionar nodos dentro de un documento XML. Es ampliamente utilizado en procesos de transformación y extracción de datos.

  • Acceso a los nodos: Permite identificar y recorrer los elementos de un XML.
  • Extracción de datos: Facilita la recuperación de información específica dentro del documento.
  • Visualización de elementos: Ayuda a estructurar y presentar el contenido de un XML.
  • Hacer cálculos: Posibilita realizar operaciones sobre los valores extraídos.

Árbol del documento XML: Tipos de nodos

En un documento XML, la estructura se organiza en forma de árbol con diferentes tipos de nodos:

  • Raíz: Es el nodo principal del documento y representa el nivel más alto. Siempre se indica con /.
  • Elemento: Contiene la información estructurada dentro del XML.
  • Atributo: Proporciona detalles adicionales sobre un elemento.
  • Texto (contenido): Representa el valor contenido en un elemento.

Ejemplo de estructura XML:

<concesionario>
  <coche matricula="6732KPC">
    <marca>Seat Ateca</marca>
    <combustible>Gasolina</combustible>
    <precio>23000</precio>
    <precio_con_iva tipo="16%">26680</precio_con_iva>
  </coche>
  <coche matricula="8280KWT">
    <marca>Volkswagen Golf</marca>
    <combustible>Diesel</combustible>
    <precio>25000</precio>
    <precio_con_iva tipo="16%">29000</precio_con_iva>
  </coche>
</concesionario>

Sintaxis XPath

Una expresión XPath es una cadena de texto utilizada para localizar nodos dentro de un árbol XML. Su estructura consta de:

  • Eje: Permite seleccionar elementos o atributos dentro del documento.
  • Predicado: Aplica condiciones a la selección para afinar la búsqueda.
  • Selección de nodos: Define el resultado final de la consulta.

Ejemplo de sintaxis:

/etiqueta1/etiqueta2/[@atributo="valor"]/etiqueta3

Ejemplo aplicado:

/concesionario/coche[@matricula="8280KWT"]/color/text()

  • Eje: /concesionario/coche
  • Predicado: [@matricula="8280KWT"]
  • Selección de nodos: /color/text()

Notas sobre el eje XPath

XPath permite navegar por un documento XML seleccionando nodos en función de su ubicación y relaciones dentro del árbol. Existen diferentes tipos de ejes que ayudan a recorrer los elementos y atributos de manera eficiente.

SINTAXIS DESCRIPCIÓN
nombre_nodo Selecciona todos los nodos con ese nombre.
/ Selección de nodos desde el nodo raíz.
// Selecciona nodos desde el nodo actual sin importar dónde se encuentren.
. Selecciona el nodo actual.
.. Selecciona el padre del nodo actual.
@ Selecciona atributos.
| Permite indicar varios caminos.

Notas sobre el predicado XPath

En XPath, los predicados se utilizan para filtrar nodos dentro de una selección. Se colocan entre corchetes [ ] y permiten restringir los resultados con condiciones específicas.

RESTRICCIÓN DESCRIPCIÓN
número Cuando se obtiene varios nodos, se muestra el indicado con el número.
@atributo Selecciona los nodos que tienen el atributo especificado.
función Funciones que se pueden aplicar a los nodos obtenidos. Existen funciones de:
- Manipulación de datos: position(), last(), etc.
- Cadenas de caracteres: concat(), substring(), etc.
- Operaciones numéricas: sum(), count(), etc.
- Funciones booleanas: not(), etc.
- Funciones de fecha: current-date(), etc.
condición Selecciona los nodos que cumplen una condición. La condición se puede aplicar sobre atributos o valores de los elementos.
Se pueden utilizar operadores lógicos (and, or, not), aritméticos (+, -, *, div, mod) y de comparación (=, !=, <, >, <=, >=).

Notas sobre selección de nodos

XPath permite seleccionar diferentes tipos de nodos en un documento XML. La selección puede aplicarse a elementos, atributos o contenido de texto dentro del árbol XML.

SELECCIÓN DESCRIPCIÓN
node() Selecciona todos los hijos (elementos o texto).
text() Selecciona únicamente el texto contenido y no muestra las etiquetas.
* Selecciona todos los hijos (solo elementos).
@* Selecciona todos los atributos del nodo.

Resumen XPath

  • Selecciona nodos: Permite navegar y seleccionar nodos.
  • Sintaxis jerárquica: Usa rutas (/ y //).
  • Filtros avanzados: Soporta condiciones con predicados ([]).
  • Búsqueda flexible: Puede buscar por etiquetas, atributos (@atributo).
  • Funciones integradas: contains(), starts-with() y text().
  • Rutas absolutas y relativas: Admite rutas desde la raíz o el nodo .
  • Compatibilidad con herramientas: Usado en navegadores.
  • Escalable: Documentos XML/HTML de cualquier tamaño.
  • Universal: Es un estándar del W3C, múltiples tecnologías.
