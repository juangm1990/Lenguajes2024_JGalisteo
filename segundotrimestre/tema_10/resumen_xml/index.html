Resumen de XML


¿Qué es XML (Extensible Markup Language)? Es un lenguaje de marcado de propósito general definido por el W3C. El propósito principal del lenguaje es compartir datos a través de diferentes sistemas, como Internet. Es similar a HTML y no está predefinido: "Las etiquetas se definen y se crean".
