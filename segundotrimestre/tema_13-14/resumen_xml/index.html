Resumen Tema 13 y 14: Transformaciones XSL I y II.


Transformaciones XSLT es un lenguaje de transformación que permite convertir documentos XML en otros formatos, como HTML, XML o texto. Utiliza una hoja de estilo XSLT para aplicar reglas de transformación a los datos XML y generar el resultado deseado. Con XSLT, puedes formatear, filtrar y reorganizar el contenido de un archivo XML de manera flexible y dinámica.

Transformación XSLT

Documento con las reglas. Es el esquema XML.

  • Lenguaje declarativo: Iteraciones, variables condicionales...
  • Basado en patrones y reglas
  • XSLT - eXtended Stylesheet Language Transformations
  • Son documentos XML: Bien formados
  • Son hojas de estilo: Ampliación de CSS
  • Transformación es visual y estructural
  • Transforma XML en otro documento
  • Usa XPath y XQuery

¿Por qué usar XSLT?

  • Presentación de datos (HTML + CSS)
  • Transformación de datos XML
  • Generar informes
  • Integración de sistemas
  • Procesar un documento XML

<xsl:output method="xml" indent="yes"/>

<xsl:mode on-no-match="shallow-copy"/>

Transformaciones XSLT I

Proceso de transformación. Plantillas:

  • El procesador analiza el documento y construye el árbol del documento.
  • El procesador recorre el árbol del documento desde el nodo raíz.
  • En cada nodo recorrido, el procesador aplica o no alguna plantilla.
  • Cuando el procesador ha recorrido el árbol, se ha terminado la transformación.

El atributo match indica los elementos afectados por la plantilla y contiene una expresión XPath.

XSL-FO

XSL-FO (eXtensible Stylesheets Language Formatting Objects) es un estándar utilizado para la transformación de documentos XML a un formato de presentación, generalmente para impresión o visualización en pantalla. Se utiliza para especificar cómo se deben formatear los datos de origen XML para su presentación.

  • Son transformaciones que tienen un archivo diferente a HTML o XML.
  • Se especifica cómo queremos que se formateen los datos de origen XML para poderlos presentar en papel o en pantalla.
  • Formatos como:
    • PDF
    • SVG
    • RTF
